# Calculadora IMC em Python

Calculadora que mede o Índice de Massa Corporal, conforme a classificação.

## 🚀 Funcionalidades

- Peso / Altura
- Classificações
- Dicionário 

## 📦 Tecnologias utilizadas

- ![Python]( https://img.shields.io/badge/Python-white?logo=python)
- ![Git]( https://img.shields.io/badge/Git-white?logo=git)

## 💻 Como rodar o projeto

### Pré-requisitos

- ![VS Code]( https://img.shields.io/badge/VsCode-white?logo=visual-studio-code)
- Python Extension

### Passo a passo
1. Abrir IDE
2. Rodar o projeto.
```bash
git clone https://github.com/npresenca/calculadora-imc-python.git
cd calculadora-imc-python

python imc.py
python imc_dic.py
```
